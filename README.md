## วิธีรันโปรแกรม
1. เอาไฟล์ `.java` ทั้ง 8 ไฟล์ไปรวมไว้ในโฟลเดอร์เดียวกัน
2. เปิด Terminal หรือ cmd แล้ว cd เข้าไปที่โฟล์เดอร์นั้น
3. พิมพ์คำสั่งคอมไพล์: `javac *.java`
4. สั่งรันโปรแกรม: `java Main`
   (ตอนรันครั้งแรก ระบบจะแอบสร้างข้อมูลตัวอย่างกับไฟล์ .csv ขึ้นมาให้เ จะได้เอาไว้เทสยืมคืนได้ทันที)

## โครงสร้างคลาสหลักๆ
#### คลาสกลุ่มข้อมูล (Data Classes):
*User: ใช้สำหรับเก็บข้อมูลสมาชิกและรายการสื่อที่สมาชิกคนนั้นกำลังยืมอยู่
* LibraryItem (Abstract Class): ทำหน้าที่เป็นคลาสแม่ (Superclass) เพื่อเก็บข้อมูลพื้นฐานของสื่อ เช่น รหัส, ชื่อเรื่อง, จำนวนคงเหลือ
* PhysicalBook และ DigitalMedia (Subclasses): คลาสลูกที่ทำการสืบทอด (Inheritance) มาจาก LibraryItem เพื่อแยกประเภทของสื่อให้ชัดเจน
#### คลาสกลุ่มจัดการระบบ (Manager Class):
* PhysicalBook และ DigitalMedia (Subclasses): คลาสลูกที่ทำการสืบทอด (Inheritance) มาจาก LibraryItem เพื่อแยกประเภทของสื่อให้ชัดเจน
#### คลาสกลุ่มส่วนติดต่อผู้ใช้ (UI Classes):
* LibraryMenu: ทำหน้าที่แสดงผลเมนูโต้ตอบกับผู้ใช้งานทาง Console รับค่าคำสั่งจากคีย์บอร์ด และดักจับข้อผิดพลาด (Error Handling) เช่น การพิมพ์ตัวอักษรลงในช่องที่ต้องการตัวเลข
* Main: คลาสเริ่มต้นสำหรับการรันโปรแกรม

## ฟีเจอร์การใช้งาน
- ระบบสมาชิก: สมัครใหม่และแก้ชื่อได้ (ดักจับชื่อซ่ำไว้แล้ว)
- จัดการสื่อ: เพิ่มสื่อ (หนังสือ/สื่อดิจิตอล), ค้นหาชื่อ, หรือลบทิ้งได้
- การยืม: ระบบจะตัดสต๊อกให้อัตโนมัติ และจำกัดโควต้าให้ยืมได้ไม่เกิน 4 ต่อคน
- การคืน: ใช้เรื่อง Polymorphism มาช่วยคิดค่าปรับตอนคืนล่าช้า
- สถิติ:รายการทั้งหมด(หนังสือ/สื่อดิจิตอล) มีเมนูโชว์ว่าหนังสือเล่มไหนโดนยืมบ่อยสุด กับเชคได้ว่าใครกำลังยืมเล่มไหนค้างอยู่บ้างครับ
